Event Triggers¶

To enhance the interactivity of a VR experience, it is useful to use event triggers. These allow you to trigger W3DActions when particular events take place in virtual space. For instance, you might make an object appear only when the user is standing in particular spot in 3D space or make an object move around whenever the user tries to look directly at it.
