Making Things Happen
====================

In order to animate objects and cause interactive changes in virtual space, you
will need to use the W3DAction class and its subclasses. W3D actions may be
triggered in one of three ways:

1. :ref:`Links <pyw3d_docs_links>`
2. :ref:`Timelines <pyw3d_docs_timelines>`
3. :ref:`Event Triggers <pyw3d_docs_triggers>`
